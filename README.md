# 🌍 Climate Trend Analyzer

## 📌 Project Overview
This project analyzes climate data to identify trends, seasonal patterns, anomalies, and future predictions using Python.

## 🚀 Features
- Temperature Trend Analysis
- Rainfall & Humidity Analysis
- Anomaly Detection
- Future Temperature Forecasting
- Data Visualization
- Streamlit Dashboard

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit

## 📊 Outputs
- Temperature Trend Graph
- Rainfall Graph
- Humidity Graph
- Anomaly Detection Graph
- Forecast Graph

## ▶️ How to Run

```bash
python generate_data.py
python main.py
streamlit run app.py
